# TimePiece - Watch Selling Website

A modern, responsive website for selling premium watches with a beautiful UI and smooth user experience.

## Pages

1. **Home (index.html)** - Landing page with hero section, features, and featured products
2. **About Us (about.html)** - Company story, mission, and values
3. **Products (products.html)** - Product catalog with filtering functionality
4. **Customer Reviews (reviews.html)** - Customer testimonials and ratings
5. **Login/Sign Up (login.html)** - User authentication with tab switching

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern and beautiful UI with gradient designs
- ✅ Smooth animations and transitions
- ✅ Product filtering by category
- ✅ Form validation for login/signup
- ✅ Mobile-friendly navigation menu
- ✅ Interactive elements (add to cart, filters, tabs)

## How to Use

1. Open `index.html` in your web browser
2. Navigate through the pages using the navigation menu
3. All pages are fully functional with interactive features

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)

## Customization

You can easily customize:
- Colors in the `:root` variables in `styles.css`
- Products in `products.html`
- Reviews in `reviews.html`
- Company information in `about.html`
